\begin{frame}[fragile]{HALD CEMENT - MODEL ANALYSIS and DETAILS}
\protect\hypertarget{hald-cement---model-analysis-and-details}{}
\begin{itemize}
\item
  An F statistic is a value you get when you run an ANOVA test or a
  regression analysis to find out if the means between two populations
  are significantly different.
\item
  It's similar to a T statistic from a T-Test;
\item
  A T-test will tell you if a single variable is statistically
  significant and
\item
  an F test will tell you if a group of variables are jointly
  significant.
\item
  \url{https://www.statisticshowto.com/probability-and-statistics/f-statistic-value-test/}
\item
  Regression NULL Hypothesis: All of the regression coefficients are
  equal to zero
\item
  Tip: P-value is High ,NUll will Fly
\item
\begin{verbatim}
 P-value is low, NULL will blow
\end{verbatim}
\item
  T-Test in Regression
\item
  \url{https://www.statology.org/t-test-linear-regression/}
\end{itemize}

\begin{frame}{Influence points analysis}
\protect\hypertarget{influence-points-analysis}{}
\begin{itemize}
\item
  n= 4 and k= 9
\item
  A Point is an influence point if

  \begin{itemize}
  \item
    COOK's D-statistic(D) \textgreater{} 1
  \item
    DFBETAS \textgreater{}
    \({2 \over \sqrt{n}} = {2 \over \sqrt{4}} = 1\), i.e DFBETAS
    \textgreater{} 1
  \item
    DFFITS \textgreater{}
    \(2 * \sqrt{ k \over n} = 2 * \sqrt{9 \over 4} = 3\) , i.e DFFITS
    \textgreater{} 3
  \end{itemize}
\item
  INFLUENCE POINT
\item
  Point 3 (as per DFBETAS). None as per DFFITS and cook's distance D.
\item
  INVESTIGATION
\item
  Point 3 is within 2 sigma limits of X1,X2,X3 and X4. So retain it
\end{itemize}
\end{frame}

  \begin{itemize}
  \tightlist
  \item
    H0: \(\rho\) = 0 ; i.e No relationship between the variables.

  \end{itemize}

  (p=0.6),X1,X4(p=0.4),X3,X4 (p=0.9)
\item
  Correlation exists : Accept H1 : X1,X3 (\textless{} 0.05);X2,X4
  (\textless{} 0.05)
